## English

### About the project

Personal portfolio and business card website built with React + Vite. This is both a project I'm actively developing and a place where I present myself — my skills, projects, and capabilities as a developer.

### Tech stack

- **React** — UI components
- **Vite** — build tool and dev server
- **React Router DOM** — client-side navigation
- **CSS / custom styles** — no UI framework

### Project structure

```
src/
├── components/       # reusable UI pieces
│   ├── Navbar.jsx
│   ├── Badge.jsx
│   └── ProjectCard.jsx
├── pages/            # full page views
│   ├── Home.jsx
│   ├── About.jsx
│   ├── Projects.jsx
│   └── Contact.jsx
├── App.jsx           # routing setup
└── main.jsx          # entry point
```

### Getting started

```bash
# Clone the repo
git clone https://github.com/artmenxxx/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
